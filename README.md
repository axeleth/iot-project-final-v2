# Indoor Environment Monitor -- Dashboard

A Streamlit dashboard that visualises indoor sensor data (temperature, humidity, light) alongside outdoor weather data, and performs cross-correlation analysis to measure how quickly an apartment responds to outdoor weather changes.

## Prerequisites

- Python 3.9 or higher
- pip

## Setup

1. Navigate to the dashboard directory:

   ```bash
   cd dashboard
   ```

2. Install the required Python packages:

   ```bash
   pip install -r requirements.txt
   ```

   This installs: streamlit, pandas, plotly, numpy, scipy

3. Make sure the data CSV files are present at:

   ```
   ./data_collection/data/indoor_sensor_data.csv
   ./data_collection/data/outdoor_weather_data.csv
   ```

   These files are generated by the data collection system (ESP32 sensor node + OpenWeatherMap API poller). Pre-collected data is included in the repository.

## Running the Dashboard

```bash
streamlit run dashboard.py
```

The dashboard will open in your browser at `http://localhost:8501`.
